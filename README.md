## Тематика сайта, название

Тематика: Идея реализовать веб-ресурс с коллекцией проверенных моделей, оптимизированных для аддитивных технологий(https://ru.wikipedia.org/w/index.php?title=3D-%D0%BF%D1%80%D0%B8%D0%BD%D1%82%D0%B5%D1%80&spm=a2ty_o01.29997173.0.0.61de55fbYowESc), чтобы снизить порог входа для начинающих пользователей FDM-принтеров и минимизировать риск неудачных первых печатей.
Название: "«Каталог простых 3D-моделей для новичков»".

## Цель и задачи сайта

Цель: Создать удобный веб-ресурс, где новички в области 3D-печати смогут быстро находить, просматривать и скачивать проверенные модели с готовыми рекомендациями по настройкам печати.

Задачи:
 1. Систематизировать коллекцию моделей по категориям, типу пластика и уровню сложности.
 2. Реализовать интуитивную навигацию с фильтрами и поиском для удобного выбора модели.
 3. Предоставить для каждой модели описание, скриншоты/3D-превью и рекомендуемые параметры слайсера.

## Целевая аудитория

Начинающие пользователи 3D-принтеров

## Содержание сайта

    Главная страница:
Сетка карточек 3D-моделей с превью-изображениями
Боковая панель или верхняя панель с фильтрами (категория, сложность, время печати, материал)
Строка поиска по названию моделей
Кнопки сортировки (по популярности, времени добавления, сложности)

    Страница модели:
3D-вьювер для осмотра модели
Описание модели и её характеристики
Блок с рекомендуемыми настройками печати
Кнопка скачивания STL-файла
Информация об авторе модели

    Страница автора:
Информация об авторе
Сетка всех моделей этого автора с превью

    Раздел «Гайд по настройке принтера»:
Пошаговые инструкции: калибровка стола, выбор филамента, базовые параметры слайсера

## Функциональные возможности

Адаптивная вёрстка.
Динамическая фильтрация каталога: Мгновенная сортировка моделей по категориям, сложности, материалу и времени печати.
Поиск моделей по названию.
Интерактивный просмотр 3D-моделей прямо в браузере.
Скачивание моделей.
Переход на страницу автора с отображением всех его работ и возможностью вернуться к основному каталогу.
Пошаговые инструкции по работе с 3D-принтером.
